# 7. 선언과 초기화를 함께 옮기기
- 보통 변수 이름에서 힌트를 얻기도 하지만 변수 초기화는 이름이 주는 의미를 더 강화 시킴
- 타입이 포함된 선언과 초기화 코드가 떨어져 있으면 읽기 어려움 -> 한참 뒤에 보면 어떤 맥락에서 선언되었는지 잊음
```
fn()
  int a
  ... 변수 a를 사용하지 않은 코드
  a = ...
  int b
  ... 변수 a를 사용할 수 있으나 변수 b를 사용하지 않는 코드
  b = ...a...
  ... 변수 b를 사용하는 코드
```
- 초기화를 변수 선언 근처로 이동하면 위에 이슈 해결
```
fn()
  int a = ...
  ... 변수 a를 사용하지 않은 코드
  ... 변수 a를 사용할 수 있으나 변수 b를 사용하지 않는 코드
  int b = ...a...
  ... 변수 b를 사용하는 코드
```
