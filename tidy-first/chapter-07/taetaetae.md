# 07. 선언과 초기화를 함께 옮기기
- as-is
  ```java
  ina a;
  ... 변수 a를 사용하지 않는 코드
  a = ...
  int b;
  ... 변수 a를 사용할 수 있으나 변수 b를 사용하지 않는 코드
  b = ... a...
  ... 변수 b를 사용하는 코드 
  ```
  
- to-be
  ```java
  ina a;
  ... 변수 a를 사용하지 않는 코드
  ... 변수 a를 사용할 수 있으나 변수 b를 사용하지 않는 코드
  int b = ... a...
  ... 변수 b를 사용하는 코드 
  ```
  
- 미리 선언/초기화 하지 말고 사용하기 직전에 선언/초기화 하자.
- 코드를 읽어 내려가는 관점에서 이해가 쉽도록
- 두려움을 느끼지 않는 수준으로 작업을 작계 진행할 것
