# 12. 도우미 호출
- 도우미(Helper)의 이름은 작동 방식이 아니라 목적에 따라 만들자.
- 예시 1
  - as-is
    ```java
    routine()
        ...그대로 두는 코드
        ...바꾸려는 코드
        ...그대로 두는 코드
    ```
  - to-be
    ```java
    helper()
      ..바꾸려는 코드
    routine()
        ...그대로 두는 코드
        helper() // 요건 11장 (비슷한 코드끼리) 와 연관되어 보임
        ...그대로 두는 코드
    ```
- 예시 2 : 시간적 결합을 표현
  - as-is
    ```java
    foo.a()
    foo.b()
    ```
  - to-be
    ```java
    ab()
      a()
      b()
    ```
